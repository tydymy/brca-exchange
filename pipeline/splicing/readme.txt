brcaParse.py is the python file with the classes to create the
correctly formatted MaxEntScan output. This output called maxEntScanList
is put into MaxEntScan. The file named maxEntOut is the out put file from MaxEnt Scan
which is also an input file for program. The MaxEntScan output is used to create a list of the MaxEntScan scores. Variants is the input file from BRCA Exchange.
